How to create or be part of a group! flutter_dash

Everything is pretty simple! Go to the group option menu and there you can choose to create the group, or if it already exists, you can give the key to a person you want to add or you can add it yourself if you go to the group menu and if you know them email address. See? It's very simple!

How to manage a group! flutter_dash

What can you do once you are part of a group? Well, it's very simple. You can edit group details, add or delete members in case you are a student. You can keep in touch with the members of the group, which you can view. Everything is very simple!
